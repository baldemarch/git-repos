#First steps#

  1. Clone repo
  2. > npm i
  3. > gulp default | open build/.
  4. double click index.html

## Alternative
  1. Repeat steps 1 and 2
  2. > gulp dev

## Test
  1. Repeat steps 1 and 2
  2. cd app/src/tests
  3. karma start
